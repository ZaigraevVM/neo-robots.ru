Разработку МГУ используют в Сибири для программирования роботов

Иркутский университет начал программировать роботов с помощью разработки МГУ

МОСКВА, 8 ноября/ Радио Sputnik. В Иркутском госуниверситете (ИГУ) используют разработку Московского государственного университета для программирования роботов, сообщает ИА IrkutskMedia.
ИГУ приобрел роботизированную учебную платформу Turtlebro – одну из разработок Научно-исследовательского института механики МГУ им. М. В. Ломоносова. Комплекс Turtlebro позволяет освоить современную робототехнику на основе Robot Operating System (ROS) – платформы для написания программного обеспечения роботов.
"Особо нас интересует управление роботами на основе навигации и картографии, машинного зрения. В ближайшее время для студентов мы разработаем практические занятия по навигации, составлению карт помещения и так далее, будем учить студентов программировать различные алгоритмы, чтобы они с помощью специальных программных кодов смогли управлять роботом", – рассказал сотрудник института математики и информационных технологий ИГУ Юрий Вшивков.
Ранее радио Sputnik рассказывало о разработке в России банкоматов на базе отечественного процессора.
